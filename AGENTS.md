# Project Overview

A full-stack platform for NGOs, volunteers, researchers, and journalists to explore survey data. Users can access extracted datasets, submit datasets (volunteers), and view datasets (NGOs). AI-powered agents provide insights from datasets. Volunteers receive ratings. Similar to Kaggle's dataset sharing model.

# Tech Stack

Next.js, Hono, Turbo.

# Target Users

NGOs, volunteers, researchers, journalists.
